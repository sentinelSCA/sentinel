# Sentinel Roadmap

This roadmap outlines the development direction for Sentinel.

## Core Platform

- Agent identity verification (Ed25519)
- Capability-based authorization
- Deterministic policy engine
- Controlled execution workers
- Human approval workflows
- Operator dashboard

## Observability

- Real-time action timeline
- AI action reports
- tamper-evident audit ledger
- replay and forensic analysis

## Security Enhancements

- agent key rotation
- capability token expiration
- policy-as-code framework
- anomaly detection

## Ecosystem

- SDK for agent integration
- infrastructure adapters
- enterprise deployment guides

## Long Term Vision

Sentinel aims to become a control plane for AI actions,
providing deterministic execution governance for autonomous systems.
