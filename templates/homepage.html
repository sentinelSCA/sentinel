Sentinel — The Firewall for AI Agents

Sentinel is an agent security control plane that evaluates actions, enforces policy, verifies execution integrity, and records tamper-evident audit trails for every decision.

Open Dashboard View Telemetry

Architecture

  • Policy decision engine for agent actions
  • Signed analyze requests with replay protection
  • Digest verification before execution
  • Append-only tamper-evident audit chain
  • Operational dashboard and queue telemetry

Capabilities

  • Agent registration, activation, and suspension
  • Risk scoring and allow / review / deny decisions
  • Human approval flow for sensitive actions
  • Execution tracking across worker queues
  • Auditability for compliance and incident response
